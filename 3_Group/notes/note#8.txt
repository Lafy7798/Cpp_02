예제 11-9 예제 안보고 풀어보기 main 부분만 카피-중요

팀: 처음에는 Point 객체를 만들어야 합니다. 
그리고 Point의 멤버 변수 x와 y를 지정하고 매개변수화 시키면 될 것 같습니다.
나: friend 키워드를 사용하여 private 멤버에 접근할수 있도록 해야 될 것 같습니다
팀: ostream& operator<< 으로 << 연산자를 오버로딩 해야 합니다
팀: 스트림에 좌표 정보를 출력해야 되는데 
stream << "(" << a.x << "," << a.y << ")"; 을 이용하여야 할 것 같습니다
 이렇게 되면 좌표 정보가 (x,y) 로 출력 될 것 같습니다
그리고 return 으로 stream을 반환해야 여러 객체를 출력 할 수 있을 것 같습니다
