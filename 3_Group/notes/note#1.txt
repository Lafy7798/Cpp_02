기계어, 어셈블리어, 고급언어란?
기계어 = 기계만 사용 가능
어셈블리어, 고급언어 = 기계어를 사람이 쓸 수 있게 편리하게 바꿈 이를 컴파일하여
기계어로 만들어 명령을 내림

캡슐화, 상속, 다형성이란?
캡슐화 = 캡슐화는 클래스라는 캡슐에 객체가 들어있는 것
상속 = 클래스의 객체를 다른 클래스에 상속하는 것
다형성 = 하나의 기능으로 여러가지 기능을 할 수 있게 함 

캡슐화 , 클래스 , 객체
캡슐화를 통해 객체를 외부로부터 보호함
외부에서의 객체를 변화시킬 수 있는 가능성을 방지함
클래스라는 틀 에서 여러 객체들을 생성해냄
클래스 = 틀
객체 = 내용물

상속성, 다형성
상속성 = 이전에 기능을 다시 정의하지 않고 편리하게 사용 가능
필요한 기능을 재정의하지 않고 그 기능이 있는 클래스 그대로 가져와서 사용 
다형성 = 각 개체들에 맞게 함수를 재정의 함
필요한 기능을 상황에 맞게 재정의 해서 사용 함
둘다 코드를 간략하게 만듬
